Backend for Frontend Sample SPA

Demonstrates using a Reverse Proxy in a SPA hosting .NET Core application with Identity Server integration.

  • Home(Anonymous Access) Where you are now
  • Fetch data(Anonymous Access) Accesses API endpoint on SpaHost
  • Fetch User data(Authenticated Access) When the page loads it checks if the user is authenticated and if not redirects to the login page
  • Guarded Route(Hidden Authenticated Access) The link is hidden until the user authenticates. If the user tries to go to the page by typing in the URL themselves they will see nothing because it is protected by an Authentication Guard.
